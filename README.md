# Bewa

Reproductor de música gratis

## Funcionalidades

Similar a Brave Playlist, pero con las funcionalidades de:

- Filtrar por Autor, Nombre
- Artistas favoritos
- Canciones mas escuchadas
- Escuchar cualquier canción mientras se tenga conexión a internet
- Descargar ciertas canciones (disponibles sin conexión)

Además que se pueda correr en segundo plano (Brave hace eso pero no puedes cerrar la pestaña de playlist para buscar algo en el navegador)
